# AI SSH Client - 使用说明

## 🚀 快速启动（推荐）

### 开发模式启动

**方式一：双击启动（最简单）**
- 直接双击 `启动开发模式.bat` 文件即可启动

**方式二：PowerShell 启动**
- 右键 `启动开发模式.ps1` → 使用 PowerShell 运行

### 构建可执行文件

**方式一：交互式构建（推荐）**
- 双击 `构建应用.bat`，根据提示选择构建类型

**方式二：命令行构建**
```bash
# 快速打包（不生成安装包）
npm run pack

# 完整构建（NSIS 安装包 + 便携版）
npm run dist:win

# 仅构建便携版
npm run dist:portable
```

---

## 📦 构建产物说明

构建完成后，文件会输出到 `release/` 目录：

- `AI SSH Client Setup x.x.x.exe` - NSIS 安装包（推荐普通用户使用）
- `AI-SSH-Client-portable.exe` - 便携版，无需安装，双击即用

---

## 🔧 传统命令行方式（开发者）

如果你还是喜欢用命令行：

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

---

## 💡 提示

1. **首次使用**：需要先运行 `npm install` 安装依赖
2. **开发模式**：修改代码后会自动热重载
3. **构建时间**：首次构建可能需要几分钟，请耐心等待
4. **杀毒软件**：构建的 exe 文件可能会被杀毒软件误报，添加信任即可
